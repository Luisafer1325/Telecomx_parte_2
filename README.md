# Telecomx_parte_2

Predicción de Cancelación de Clientes (Churn)

Este proyecto analiza datos de clientes para identificar los principales factores asociados a la cancelación de servicios (churn). Se realizó un análisis exploratorio de datos, visualización de relaciones entre variables y la construcción de modelos de aprendizaje automático.

Se utilizaron modelos como Regresión Logística y Random Forest para predecir la probabilidad de cancelación. Los modelos fueron evaluados mediante métricas como exactitud, precisión, recall, F1-score y matriz de confusión.

Los resultados permiten identificar variables clave que influyen en la cancelación y proponer estrategias que ayuden a mejorar la retención de clientes.
